# Recall Grounds

**Recall Grounds** is a pixel art arena survival game built in **C using raylib**, centered around one satisfying mechanic — throwing and recalling your boomerang.

**Throw it to deal damage. Recall it back for a bonus hit. Survive as long as you can.**

**Status:** Released — v1.0

🎮 [Play for free on itch.io](https://stillbenja.itch.io/recall-grounds)

## Features

- Throw and recall boomerang combat with bonus damage on return hits
- Upgrade cards to build your own playstyle
- Enemies that get faster and stronger the longer you last
- Escalating difficulty with visual enemy tiers
- Highscore saving
- Built entirely in C using raylib

## Controls

| Key | Action |
|-----|--------|
| WASD | Move |
| LMB | Throw |
| RMB | Recall |
| Space | Mystic Pull |
| ESC | Pause |
| F11 | Toggle Fullscreen |

## Build

- **Language:** C
- **Framework:** raylib
- **Build System:** CMake
- **Tested on:** Windows / Linux

Compile and link against raylib using your existing setup.
